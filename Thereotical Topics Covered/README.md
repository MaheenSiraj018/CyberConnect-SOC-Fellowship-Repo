# Week 01 and Week 02 – Network Fundamentals

This folder contains an overview of the theoretical knowledge covered so far as part of the CyberConnect SOC Fellowship program.

The focus of this week was building foundational cybersecurity knowledge, including networking basics, defensive and offensive security concepts, and understanding various career paths in cybersecurity.

## Rooms Covered

- Intro to Networking  
- Defensive Security Introduction  
- Offensive Security Introduction  
- Careers in Cyber  
- Network Traffic Basics  

These rooms provided the core concepts required for understanding how networks operate and how security professionals monitor, defend, and analyze network activity.
