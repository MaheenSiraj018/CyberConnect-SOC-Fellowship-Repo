# Intro to Networking

## Overview
This room introduced fundamental networking concepts that form the foundation of cybersecurity.

## Topics Covered
- OSI Model
- TCP/IP Model
- Ports and Protocols
- IP Addressing
- How data flows across a network

## Key Takeaways
- Understanding how packets move through network layers.
- Differences between TCP and UDP.
- Importance of well-known ports (HTTP, HTTPS, DNS, SSH).
- Basic understanding of how devices communicate over a network.
