# Network Traffic Basics

## Overview
This room focused on understanding and analyzing network traffic as part of defensive security operations.

## Topics Covered
- Logs vs Full Packet Capture
- Network TAP and Port Mirroring
- HTTP and DNS traffic analysis
- NetFlow and IPFIX basics

## Key Takeaways
- Importance of packet inspection in threat detection.
- Identifying suspicious HTTP and DNS activity.
- Understanding how traffic metadata helps detect anomalies.
